블로그 글은 전부 **마크다운(Markdown)** 이라는 간단한 문법으로 씁니다.
이 글 자체가 예시이자 치트시트예요.

## 제목

`#` 개수로 제목 크기를 정합니다.

```
## 큰 제목
### 중간 제목
```

## 글자 꾸미기

- `**굵게**` → **굵게**
- `*기울임*` → *기울임*
- `~~취소선~~` → ~~취소선~~

## 목록

```
- 순서 없는 목록
1. 순서 있는 목록
```

## 링크와 사진

```
[보여줄 글자](https://www.google.com)
![사진 설명](/images/사진파일이름.jpg)
```

사진은 `public/images` 폴더에 넣고 위처럼 경로를 적으면 됩니다.

## 인용

```
> 인용하고 싶은 문장
```

> 이렇게 표시됩니다.

## 코드

백틱(`) 세 개로 감싸면 코드 블록이 됩니다. 공부 정리할 때 유용해요.

```python
def hello():
    print("안녕, 블로그!")
```
